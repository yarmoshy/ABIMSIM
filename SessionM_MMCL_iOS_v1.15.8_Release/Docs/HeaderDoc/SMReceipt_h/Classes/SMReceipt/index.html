SMReceipt

Superclass:
NSObject
Declared In:
SMReceipt.h

Introduction

Defines the data associated with a receipt uploaded by the user.


Methods

-notifyPresented

Notifies the SDK that the view containing the associated receipt data was presented to the user. Used for reporting purposes.

-notifySeen

Notifies the SDK that the view containing the associated receipt data was seen by the user. Used for reporting purposes.

-notifyTapped

Notifies the SDK that the user tapped on the view containing the associated receipt data. Used for reporting purposes.

-presentDetailsPage

Presents the content pointed to by detailsURL.
