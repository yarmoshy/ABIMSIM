Introduction

SessionM Location service interface. This is the main class in SessionM Location iOS SDK. It defines methods for all location based operations such as location and region monitoring.

refreshGeofenceService

Refreshes region monitoring based on SessionM Mobile Marketing Cloud portal configurations. 

- (void)refreshGeofenceService;
Discussion

This call made on the [SMLocationManager sharedInstance] object will refresh monitoring regions based on configurations setup by the developer in the SessionM Mobile Marketing Cloud portal. Regions monitored persist across app launches and are only cleared via stopGeofenceService. Required always-on location permission to work.

sharedInstance

Returns singleton SMLocationManger service instance with a monitored location event limit of SMLocationManagerMonitoredEventLimit. 

+ (SMLocationManager *)sharedInstance;
Return Value

SMLocationManger service object.

start

Starts location services. 

- (void)start;
Discussion

This call made on the [SMLocationManager sharedInstance] object will start location updates. If permission is denied a SMLocationManagerLocationServicesDisabled notification will be fired. This method looks for the Cocoa Key NSLocationAlwaysUsageDescription to determine whether to start always-on or while-in-use monitoring.

startGeofenceService

Starts region monitoring based on SessionM Mobile Marketing Cloud portal configurations. 

- (void)startGeofenceService;
Discussion

This call made on the [SMLocationManager sharedInstance] object will start monitoring regions based on configurations setup by the developer in the SessionM Mobile Marketing Cloud portal. Regions monitored persist across app launches and are only cleared via stopGeofenceService. Required always-on location permission to work.

stop

Stops location services. 

- (void)stop;
Discussion

This call made on the [SMLocationManager sharedInstance] object will stop location updates. This does not clear monitored regions that persist across app launches.

stopGeofenceService

Stops all region monitoring. 

- (void)stopGeofenceService;
Discussion

This call made on the [SMLocationManager sharedInstance] object will stop monitoring all regions.
