SMMessageData

Superclass:
NSObject
Declared In:
SMMessageData.h

Introduction

Defines the data associated with a message.

Discussion

Note: the developer can configure the following properties for each message through the SessionM Mobile Marketing Cloud portal.


Methods

-notifyPresented

Notifies the SDK that the view containing the associated message data was presented to the user. Used for reporting purposes.

-notifySeen

Notifies the SDK that the view containing the associated message data was seen by the user. Used for reporting purposes.

-notifyTapped

Notifies the SDK that the user tapped on the view containing the associated message data. Used for reporting purposes.
