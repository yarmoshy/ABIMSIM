Superclass:
UIViewController
Declared In:
SMReceiptUploadViewController.h

Introduction

View controller for receipt upload activity.

Discussion

An instance of this class is created and presented by the SDK when the user is asked to upload a receipt image for particular ads. The developer can optionally create an instance using the initWithAttributes: method to upload receipt images without going through the ad flow.
